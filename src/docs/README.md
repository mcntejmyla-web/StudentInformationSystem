# Student Information System Documentation

## Project Overview
This project is a Java-based Student Information System that stores and manages student records.

## Objectives
- Store student details
- Display student information
- Validate user input
- Practice Java OOP concepts

## Features
- Add Student
- View Student Records
- Search Student
- Update Student Details
- Menu Driven Interface
- Data Validation

## Technologies Used
- Java
- Scanner Class
- ArrayList

## Setup Instructions
1. Install JDK
2. Open any Java IDE
3. Compile the source code
4. Run the program
5. Follow the menu options

## Code Structure
- Student Class
- Main Class
- ArrayList for storage
- Scanner for user input

## Technical Requirements
✓ Student class with attributes

✓ Age validation

✓ Grade validation

✓ Menu-driven interface

✓ Search functionality

✓ Formatted output

✓ Error handling

✓ Data storage using ArrayList
